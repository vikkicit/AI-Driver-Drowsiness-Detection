# 🚗 AI Driver Drowsiness Detection System

An AI-powered driver drowsiness detection system that detects:

- 👁️ Eye Open
- 😴 Eye Closed
- 🥱 Yawn
- 🙂 No Yawn

using CNN Feature Extraction, HOG(Zernike-like) Features, Random Forest and Real-Time Webcam Monitoring.

## Accuracy

| Model | Accuracy |
|---------|---------|
| Eye Detection | 98.62% |
| Mouth Detection | 95.81% |
